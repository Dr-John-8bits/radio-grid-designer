Grille hebdomadaire

Édition directe à la souris

7 jours visibles, pas de 5 minutes, chevauchements autorisés pour les émissions ponctuelles. Les créneaux sélectionnés passent devant les autres pour rester manipulables.

Pas de 5 min
?
Glisse dans un jour pour créer un créneau. Attrape un créneau existant pour le déplacer. Utilise les poignées en haut et en bas pour modifier ses horaires.

Édition

Créneau sélectionné

?
Tous les champs de ce panneau modifient le créneau sélectionné dans la grille. Si rien n’est sélectionné, commence par cliquer sur un bloc ou une émission.

Profil

Répartition documentaire

?
Ce panneau synthétise la logique éditoriale de l’émission sélectionnée: type de définition, tags principaux et tags secondaires issus de la documentation du projet.

Tags

Bibliothèque de genres

?
Tous les tags importés depuis la supervision et la cartographie officielle sont consultables ici. Tu peux limiter l’affichage aux tags liés à l’émission sélectionnée.

Morceaux

Morceaux du genre

?
Cet encart affiche les morceaux correspondant au tag recherché dans la bibliothèque. Tu peux aussi cliquer un tag pour le verrouiller et voir uniquement ses morceaux.

Habillage

Jingles et réclames

?
Ces réglages pilotent la fréquence approximative des éléments d’habillage sans écraser les émissions. Tu peux définir un intervalle, un décalage et la source utilisée.

Exploitation

Runtime Liquidsoap

?
Cet encart regroupe les informations nécessaires pour générer ensuite un vrai script radio.liq : chemins système, entrée live, sorties Icecast et politique de rotation.

Aperçu

État exporté

?
Cet encart montre exactement ce qui sera exporté depuis le navigateur. Le JSON sert de format maître. Le JSONL sert de format à plat pour les traitements automatiques.
